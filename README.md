# Luaplus (ltsteamplugin)

Plugin para o cliente Steam baseado no ecossistema Millennium. Este projeto é uma modificação do plugin original criado por [madoiscool](https://github.com/madoiscool/ltsteamplugin).

## Funcionalidades e Modificações

* **Suporte Completo ao Big Picture**: Interface adaptativa otimizada para o modo Big Picture da Steam e Steam Deck.
* **Suporte a Controle**: Navegação por gamepad mapeada para o modal de configurações e ferramentas.
* **Save Manager**: Sistema de backup e restauração de save games com rotação automática dos últimos 5 backups, rollback seguro contra falhas e proteção de limites de segurança para diretórios massivos (limite de 100MB e 1000 arquivos).

## Como Instalar

1. Certifique-se de possuir o [Millennium Steam Client](https://steambrew.app/) instalado.
2. Baixe o código deste repositório.
3. Crie a pasta `ltsteamplugin` no diretório de plugins da Steam (geralmente em `C:\Program Files (x86)\Steam\plugins\ltsteamplugin`).
4. Extraia todo o conteúdo do repositório dentro desta pasta.
5. Reinicie a Steam e ative o plugin nas configurações do Millennium.
